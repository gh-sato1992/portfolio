APPLE BERRY
（スクールの課題）

サイト名 APPLE BERRY
使用言語 HTML / CSS / Sass / jQuery
レスポンシブ スマートフォン / PC
制作期間 10日
制作の目的 雑貨屋さんがオンラインで商品を販売できるようにECサイトを開設。
ターゲット 20〜40代の雑貨好きな人。主にスマートフォンを利用し、時々PCを利用する人。
デザイン 女性向けですが、男性でも問題なく利用できるデザインを意識しています。
備考 スクールの課題で制作しました。作成ページは、TOPページ、商品一覧ページ、商品詳細ページのみです。スクールでは段階的にコーディングの難易度が上がっていき、当時はこのECサイトの制作に高い壁を感じました。このECサイトの制作を通して、カルーセルやサイドバーの設置、画像をタイル状に配置していく技術を学びました。
