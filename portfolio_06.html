OKASHIYA
（架空サイト）

サイト名 OKASHIYA
使用言語 HTML / CSS / Sass / jQuery
レスポンシブ スマートフォン / PC
制作期間 3日
制作の目的 洋菓子屋さんがオンラインで商品を販売できるようにECサイトを開設。
ターゲット チョコレートやクッキーが好きな女性。プレゼント用にデザイン性のあるお菓子を求めている人。
デザイン モバイルファーストを意識したシングルページです。商品数が小規模と仮定し、サイドバーは採用せず、商品画像を一覧で見やすいようにしました。
背景色は優しい雰囲気の薄い黄色を使用し、各タイトルには派手な黄色のマーカーで下線を引いて、色のメリハリをつけました。リピーター向けに、商品一覧ページからお気入りの商品を選んで、まとめてカートに入れることができるような設計にしています。
備考 企画からコーディングまで行いました。作成ページは、TOPページと商品一覧ページのみです。今回はカートボタンの作成まで行いました。今後は実用的なカートボタンの実装ができるような技術を身につけていきたいです。
