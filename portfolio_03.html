Filmcamera's World
（架空サイト）

サイト名 Filmcamera's World
使用言語 HTML / CSS / Sass / jQuery
レスポンシブ スマートフォン / PC
制作期間 4日
制作の目的 フィルムカメラ好きとフィルムカメラに触れたことがない若い世代に向けたイベントを実施するのに伴い、フィルムカメラの魅力を広めるためのプロモーションサイトとして開設。
ターゲット メインターゲットは18〜30歳の女性層、その他フィルムカメラファン。
デザイン モバイルファーストを意識したシングルページです。フィルムカメラのイベントなので、レトロなデザインになるような色とフォントを選びました。特に色は、メインターゲット層に合わせてガーリーな雰囲気が漂う燻んだ色を採用しています。
備考 企画からコーディングまで行いました。今回は背景色を2色に分割し、レトロポップな雰囲気を目指しました。またスクロールするとページトップに戻るボタンの初実装に挑戦しました。アイキャッチ画像は、Photoshopでフィルムカメラ風に加工しました。その過程でスクールではあまり触れていない部分の画像加工の勉強もしました。
