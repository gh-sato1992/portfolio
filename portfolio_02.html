株式会社れき泉
（架空サイト）

サイト名 株式会社れき泉
使用言語 HTML / CSS / Sass / jQuery
レスポンシブ スマートフォン / PC
制作期間 7日
制作の目的 主に歴史関連の事業を行なっている企業のコーポレートサイトとして開設。企業の存在証明やビジネスチャンスの獲得、人材確保を目的とし、サイトをご覧くださった方がお問い合わせできるページを設置。
ターゲット 既存顧客と見込顧客、一般消費者、採用応募者。
デザイン 歴史関連事業の企業のイメージに合うように背景を和風にしました。会社概要、採用情報、お問い合わせページには、それぞれのページに合う雰囲気の鳥獣戯画のイラストを加えています。
フォントは視認性の高いゴシック体を採用し、見出しは太字に、本文は細字にしています。色は信頼性と堅実性を押し出させる青色を採用しています。また短期間で完成させるために、軽量CSSフレームワークのskeletonを使用しています。
備考 企画からコーディングまで行いました。今回は初めてCSSのみでのハンバーガーメニューを実装しました。試行錯誤を重ね、イメージ通り動きを与えられたので、チャレンジして良かったです。今後の課題はWordPressやPHPを学習し、お問合せフォームの実用化、記事を簡単に更新投稿できるようにすることです。
