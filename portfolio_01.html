KAWAcafe
（架空サイト）

サイト名 KAWAcafe
使用言語 HTML / CSS / Sass / jQuery
レスポンシブ スマートフォン / PC
制作期間 6日
制作の目的 友人が新しく喫茶店をオープンするのに伴い、集客のためにホームページを開設。
ターゲット メインターゲットは珈琲好きの方。その他近隣在住の方や仕事の打ち合わせに利用したいビジネスマンなど。
デザイン モバイルファーストを意識したシングルページです。フォントはクラシカルな雰囲気を演出できるものを採用し、背景色は白色と灰色でシックな雰囲気にし、アクセントカラーにクラシカルな淡い水色を採用しました。
またファーストビューは全画面表示に設定し、カルーセルで店舗の雰囲気が伝わるようにしました。各要素でfadeInのアニメーションが作動するように設定し、穏やかな雰囲気が出るようにしました。
備考 初めて企画からコーディングまで通して制作しました。将来喫茶店をオープンしたいという友人と共同で企画し、打ち合わせを通して完成に辿り着きました。実際の案件だと思って仕事に取り組める環境を作り出せたので、スクールで課題通りに制作するデザインやコーディングよりも良い経験になりました。
